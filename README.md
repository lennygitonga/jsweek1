# Personalized-Webpage
This is a personalized webpage created to show the user input(name and age) and calculate their age in months. The webpage can also determine whether one is an adult or a minor so as to know wheether it can display certain content or not.


## About
The project is done by Lenny Muriithi


 ## Features
 ### Form
 The form is a section in the page that collects the users data which is the name and the age.
